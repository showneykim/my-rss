IDE 설정 (Settings/Preferences) 을 열어 Ensure every saved file ends 을 검색한다. 아래와 같이 Ensure every saved file ends 항목을 체크한다. 이렇게 하면 파일을 저장할때마다 항상 newline을 추가한다. 다만, 이 방법을 추천하진 않는다. 웬만하면 아래 2번째 방법을 사용하자. 2.
